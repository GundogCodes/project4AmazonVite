### Routing

- USER ROUTES

| **HTTP Method** | **End Point**    |    **Action**     |
| :-------------- | :--------------: | ----------------: |
| POST            | /user            | Creates user      |
| POST            | /user/login      | Logs in user      |
| POST            | /user/logout/:id | Logs out user     |
| DELETE          | /user/:id        | Delete user       |
| PUT             | /user/:id        | Update user       |
| GET             | /user/:id        | Get user info     |
